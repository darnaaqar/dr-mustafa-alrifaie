# dental_clinic_app

Premium Futuristic Dental Clinic App for Dr. Mustafa Al-Rifai

## Features

- Bilingual support (Arabic/English)
- Service booking system
- Gallery showcase
- Doctor information
- Contact information
- Supabase integration
